# Credit-Card-Fraud-Detection-Systwm
A complete end-to-end machine learning pipeline for fraud detection using XGBoost, SMOTE, Bayesian Optimization, and Threshold Optimization.

Project Overview

This project implements a production-ready fraud detection system using the PaySim synthetic financial dataset.
It includes:

Automated feature engineering

Full preprocessing pipeline

SMOTE-based oversampling

Bayesian hyperparameter optimization

Classification threshold optimization

Comprehensive evaluation metrics

Save/load model utilities

Organized project structure
